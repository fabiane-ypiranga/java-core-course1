# ☕ Java Core Course

Projeto desenvolvido para praticar conceitos fundamentais de Java, começando pelos conceitos básicos até Programação Orientada a Objetos (POO).

## 📚 Conteúdos praticados

### Variables

* Criação de variáveis
* Tipos de dados (`String`, `int`, `double`)
* Conversão de tipos (`TypeCasting`)

### Control Flow

* Estruturas condicionais (`if/else`)
* Laços de repetição (`for` e `while`)

### Object-Oriented Programming (OOP)

* Criação de classes e objetos
* Atributos e métodos
* Encapsulamento
* Simulação de objetos (`Person`, `Car` e `Account`)

## 🛠 Tecnologias utilizadas

* Java
* IntelliJ IDEA
* Git & GitHub

## 🎯 Objetivo do projeto

Praticar lógica de programação e fundamentos do Java através de exercícios organizados por tema.
